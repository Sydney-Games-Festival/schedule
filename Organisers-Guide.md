# Organisers Guide

This file contains a suggested draft you can send to external organisers so they
can update their own Sydney Games Festival submission without creating duplicate
entries.

## Suggested Message To Organisers

Subject: How to update your Sydney Games Festival event submission

Hello,

If you need to update your Sydney Games Festival event submission, please use
the edit link from the confirmation email you received after you first
submitted the form.

Using the edit link updates your existing submission. Please do not submit a
second form just to make changes.

How to update your submission:

1. Open the confirmation email from your original form submission.
2. Click the link to edit your response.
3. Update your event details and submit the form again.

You can use that edit link to update details such as:

- event name
- organisation name
- description and marketing blurb
- planned date, time, and duration
- location
- audience and game type
- ticket or information URL
- thumbnail URL

Important:

- Please keep the confirmation email until the festival ends on Sunday,
  October 18, 2026.
- After you open the edit page, save or bookmark the edit link somewhere easy
  to find again.
- If you cannot find the confirmation email, contact the festival team and we
  can help you work out the next step.
- Updating your submission changes your event record, but public schedule
  changes may still depend on festival team review and publishing.

Thank you,

Sydney Games Festival team

## Internal Setup Checklist

Before sending the message above, confirm that the form is configured to:

- allow response editing
- collect responder email addresses
- send responders a copy of their response, ideally set to `Always`

It is also worth adding a short reminder to the form confirmation message:

`Please keep this confirmation email so you can edit your submission later.`
